# Mein Formular  
Ein interaktives Webformular, entwickelt mit HTML, CSS und JavaScript.

## 🎯 Überblick
„Mein Formular“ ist ein einfaches, responsives Webformular zur Erfassung und Verarbeitung von Nutzereingaben.  
Das Projekt demonstriert grundlegende Formularvalidierung, dynamische Rückmeldungen und eine klare, benutzerfreundliche Oberfläche.

## 🚀 Funktionen
- Responsives Layout  
- Validierung von Eingabefeldern  
- Dynamische Fehlermeldungen und Bestätigungen  
- Barrierearme und übersichtliche Gestaltung  

## 🧩 Verwendete Technologien
- **HTML5** – Struktur  
- **CSS3** – Layout & Design  
- **JavaScript (ES6)** – Interaktivität & Validierung
